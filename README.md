# Ticket Booking Web Application

This is a React-based web application developed for booking tickets for a college event at Vel Tech University.

## Event Details
- Event: APTI MIND 2K26
- Institution: Vel Tech University
- Department: Computer Science & Engineering
- Venue: Main Auditorium, Campus 2

## Features
- Display event details
- Ticket booking with validation
- Prevent overbooking
- Dynamic ticket availability update
- Unique Ticket ID generation
- Ticket tracking using Ticket ID
- Indian currency support (₹)

## Technologies Used
- React JS
- JavaScript
- CSS
- Vite

## How to Run
1. Install dependencies:
   npm install

2. Run the project:
   npm run dev
